Retinal Image Management System


The work was done under the following Air Force Research Laboratory (AFRL) SBIR

Contract: FA8650-14-M-6558
Topic: AF141-028

It is an mock-up website that represents a potential web interface for a Retinal Image Management System. It is built using the Flask Framework in Python. There are 5 pages of content:

All functionality is not fully implemented, this website is purely for demonstration only.

Browse
- Select a patient, review their recent history
Import
- Select Images from the local disc to upload
Register
- Demonstrate the result of the registration
- Uses the Pikachoose.js library
Compare
- Demonstrate the result of the registration
- Uses the TwoFace.js library
Analyze
- Demonstrate the result of Change detection over two sets of sample images


The above image consists of a series of Color Fundus images which have been stitched into a picture of the entire retina.
