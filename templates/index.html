Retinal Image Management System


The following work was done under the following Air Force Research Laboratory (AFRL) SBIR

Contract: FA8650-14-M-6558
Topic: AF141-028

Not all functionality is fully implemented, this website is purely for demonstration only.

This is a mock-up website that represents a potential web interface for a Retinal Image Management System. It is built using the Flask Framework in Python. There are 5 pages of content:

Browse
- Select a patient
Manage
- Select Images from the local disk to upload and select images for image registration
Review
- Demonstrate the result of the registration
- Uses the Pikachoose.js library
Compare
- Demonstrate the result of the registration over two sets of sample images
- Uses the TwoFace.js library
Analyze
- Demonstrate the result of Change detection over two sets of sample images


The image above consists of a series of Color Fundus images which have been stitched into a picture of the entire retina.
